# Todo App - Chiroyli Vazifa Menejeri

Zamonaviy va chiroyli dizaynga ega bo'lgan vazifa boshqaruv ilovasi. Ushbu ilova sizga kundalik vazifalaringizni samarali boshqarishga yordam beradi.

## Xususiyatlari

- **Chiroyli dizayn** - Zamonaviy gradient fon va yumshoq animatsiyalar
- **To'liq funksional** - Vazifalarni qo'shish, tahrirlash, o'chirish va bajarilgan deb belgilash
- **Filtrlar** - Barcha, kutilayotgan va bajarilgan vazifalarni ko'rish
- **Statistika** - Jami, bajarilgan va kutilayotgan vazifalar soni
- **Mahalliy saqlash** - Ma'lumotlarni brauzerda saqlaydi
- ** responsiveness** - Mobil va desktop qurilmalarda ishlaydi
- **Uzbek tilida** - To'liq uzbek tilida interfeys
- **Bildirishnomalar** - Harakatlar uchun chiroyli bildirishnomalar

## Foydalanish

1. Ilovani brauzeringizda oching
2. Yangi vazifa qo'shish uchun matn maydoniga yozing va "Qo'shish" tugmasini bosing
3. Vazifani bajarilgan deb belgilash uchun chapdagi doirani bosing
4. Vazifani tahrirlash uchun "Tahrir" tugmasini bosing
5. Vazifani o'chirish uchun "O'chir" tugmasini bosing
6. Filtrlardan foydalanib vazifalarni turli usullarda ko'ring

## Qisqa yo'llar (Keyboard Shortcuts)

- **Ctrl/Cmd + Enter** - Formani yuborish
- **Esc** - Tahrirni bekor qilish

## Texnologiyalar

- **HTML5** - Zamonaviy semantik markup
